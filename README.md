## ARM 기반 간호 어시스턴트 로봇


## 📌 프로젝트 개요
본 프로젝트는 **Turtlebot4**과 ROS 2 Humble를 활용하여 병원 내 간호사의 반복적 업무 부담을 줄이기 위해, 약품 전달과 환자 생체징후 확인을 지원하는 로봇을 개발하는 팀 프로젝트입니다.
